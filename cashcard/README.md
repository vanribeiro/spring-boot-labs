# Cash Card - REST API with Spring Boot

Repository containing the lab from the **Building a REST API with Spring Boot** course, by [Spring Academy](https://spring.academy/courses/building-a-rest-api-with-spring-boot).

## About the project

This project consists of a REST API for managing **Cash Cards** — prepaid cards that hold an associated balance (amount). Throughout the course, concepts such as Test-Driven Development (TDD), JSON serialization/deserialization, and best practices for building RESTful APIs with Spring Boot are covered.

## Technologies used

- **Java 21**
- **Spring Boot 4.0.6**
- **Spring Web MVC** — REST API construction
- **Gradle 9.4.1** — build and dependency management
- **JUnit 5** — testing framework
- **AssertJ** — fluent test assertions
- **Jackson (via `@JsonTest`)** — JSON serialization and deserialization

## How to run

```bash
./gradlew bootRun
```

## How to run the tests

```bash
./gradlew test
```
