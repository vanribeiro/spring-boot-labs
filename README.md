# Spring Boot Labs

This repository contains labs, study projects, and course exercises focused on learning and practicing with Spring Boot.

## Technologies Used

- **Java**: The primary programming language for all projects in this repository.
- **Spring Boot**: A powerful framework for building stand-alone, production-grade Spring-based applications with minimal configuration.
- **Gradle**: Used as the build automation tool for managing dependencies and building the projects.

## About This Repository

This repository is dedicated to my personal learning journey with Spring Boot. Here you will find code samples, experiments, and exercises from various courses and self-study sessions. The structure may include multiple projects, each exploring different aspects of Spring Boot and related technologies.

Feel free to explore the code and use it as a reference for your own learning!
